HTML

  • Is used to create the structure of a webpage.
  • HTML elements are usually composed of an opening tag and a closing tag represented with angle brackets.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The style element is used to define the CSS styles for a document.
  • The base element specifies the base URL for a webpage.
  • The img element embeds images in the HTML document.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.

CSS

  • Stands for Cascading Style Sheets.
  • It is used to describe the presentation of a website that's created with HTML elements.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Inline CSS is used to style an element directly in the HTML file
  • Internal style sheet contains CSS rules for the webpage in the HTML file head section.
  • External CSS style sheet keeps all CSS rules in a separate file.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds all the changes made in the current working branch to the staging area
  • git commit -m "description": commits files with a message that describes what the commit contains
  • git pull origin main: fetches commits from the main branch in GitHub and merges into the locally checked out branch
  • git push origin branch-name: pushes the changes made in a local branch to a remote repository
